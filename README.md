# shortener
a link shortener made in node.

## usage
clone the repo:
```bash
git clone <this repo> && cd shortener
```
install the dependencies:
```bash
pnpm install # or npm, yarn, etc.
```
run it:
```bash
pnpm start # or npm, yarn, etc.
```
